## HTTP

### Request Structure

| Method | URL or pathname                | HTTP version | headers | message body |
| ------ | ------------------------------ | ------------ | ------- | ------------ |
| GET    | //www.dcs.qmw.ac.uk/index.html | HTTP/1.1             |         |              |

- **Method**
	- nature of request
- **URL**
	- resource - object of request
- **HTTP Version**
- **Headers**
	- additional info
	- Host:\<hostname>, auth information, content type options 
- **Body**
	- Request data
	- document being uploaded
